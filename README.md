# claude-skills
Personal Claude configuration — skills and commands built by Yael Navon.
## skills/
Triggered automatically by Claude when relevant — no need to call them explicitly.
- **query-builder** — breaks down any SQL or logical data question step by step
## commands/
Slash commands triggered manually (e.g. `/morning-agent`).
- **morning-agent** — daily briefing: news, open tasks, and job search
- **mentor** — analyst mentor for analysis practice
- **start-analysis** — walks through yael's personal analysis guidebook step by step
